# Health Recommendation Engine

This skill provides personalized health suggestions based on the user's:
- Age
- Weight
- Fitness goal
- Diet preference
- Allergies
- Lifestyle

When prompted to give a health recommendation, use the user's profile information along with the nutrition information of the food item they are consuming to provide a personalized, actionable suggestion.
